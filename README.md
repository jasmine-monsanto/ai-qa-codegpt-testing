# AI QA CodeGPT Testing Project

## Overview
This project evaluates CodeGPT in VS Code for AI-assisted QA testing, focusing on workflow behavior, limitations, and usability in a freemium environment.

## Full Report
Detailed analysis is available in: AI_QA_Report.md

## Key Focus Areas
- AI-generated test case evaluation
- Workflow usability testing
- Freemium tool limitations
- QA defect observation

## Key Finding
CodeGPT workflow is limited due to freemium restrictions blocking full chat functionality after login.
